AppStream 2.0

What?

AppStream 2.0 is a fully managed non-persistent desktop and application virtualization service that allows your users to securely access the data, applications, and resources they need, anywhere, anytime, from any supported device.

https://aws.amazon.com/appstream2/

AppStream 2.0

Why?

  • Shared devices without admin
  • Devices with insufficent resources (e.g. for Unity)
  • Remote delivery of applications
  • Secure applications
  • Legacy applications
  • Graphically intense applications
